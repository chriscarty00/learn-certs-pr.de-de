# <a name="microsoft-certified-trainer"></a>Microsoft Certified Trainer

## <a name="become-a-microsoft-certified-trainer"></a>Werden Sie ein Microsoft-zertifizierter Trainer

Microsoft Certified Trainer (MCTs) sind die führenden technischen und didaktischen Experten zu Microsoft-Technologien. Schließen Sie sich dieser exklusiven Gruppe von weltweiten Microsoft Technical Training Professionals an und profitieren Sie von den Vorteilen der MCT Schulungszertifizierung und Mitgliedschaft.

Als MCT erhalten Sie exklusive Vorteile wie den Zugriff auf die komplette Bibliothek der offiziellen Microsoft Schulungs- und Zertifizierungsprodukte, erhebliche Rabatte auf Prüfungen, Bücher und Microsoft-Produkte. Darüber hinaus können Sie die Microsoft Readiness-Ressourcen nutzen, um Ihre Ausbildung zu verbessern und mit anderen MCT-Mitgliedern in einem Online-Community-Forum zusammenzuarbeiten. Sie erhalten auch Einladungen zu exklusiven Microsoft- und lokalen MCT-Community-Veranstaltungen.

Als Microsoft Certified Trainer müssen Sie für jeden Kurs, den Sie unterrichten möchten, eine Microsoft-Zertifizierung erlangen, die Ihre Erfahrung und Ihr Wissen zu dem entsprechenden Microsoft-Produkt oder der Microsoft-Lösung nachweist.

Beim Microsoft Certified Trainer-Programm handelt es sich um eine jährliche Mitgliedschaft. Sie müssen die Programmanforderungen erfüllen, einen Online-Antrag ausfüllen und eine jährliche Gebühr zahlen, um Ihre Ausbildungszertifizierung für das kommende Jahr zu erneuern.

## <a name="program-benefits"></a>Programmleistungen

Als Microsoft Certified Trainer erhalten Sie Zugriff auf ein personalisiertes Dashboard, in dem Sie eine Vielzahl von nützlichen MCT-Ressourcen finden, Ihr MCT-Zertifikat herunterladen und die für Sie verfügbaren Rabatte einsehen können!

Darüber hinaus haben Sie Zugriff auf [MCT Central](https://www.microsoft.com/en-us/learning/mct-central.aspx), was MCTs hilft, über die neuesten Nachrichten auf dem Laufenden zu bleiben, Stellen zu finden, mit anderen MCTs auf der ganzen Welt in Kontakt zu treten oder Ihre MCT-Regionalleitung zu kontaktieren.

[Erkunden Sie alle Leistungen für MCTs.](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE23o0Z)

## <a name="requirements-to-become-a-mct"></a><a name="requirements-to-become-mct"></a> Anforderungen für die Zulassung als MCT

Das MCT-Programm richtet sich an Trainer auf der ganzen Welt, die aktiv Schulungen zu Microsoft-Technologien anbieten. Wenn Sie über eine entsprechende Zertifizierung oder Qualifizierung verfügen und leidenschaftlich gern Schulungen zu Microsoft-Technologien geben, sollten Sie sich für das Programm bewerben.

Ausführliche Informationen zu den MCT-Programmanforderungen finden Sie im MCT Program Guide und im MCT Program Agreement. Beides finden Sie im Abschnitt [Bewerbungsprozess](#application-process) auf dieser Seite.


### <a name="mct-program-entry-requirements"></a>Einstiegsanforderungen für das MCT-Programm

- Besitz einer aktuellen, für das MCT-Programm infrage kommenden Microsoft-Zertifizierung oder -Qualifizierung (Eine Liste qualifizierender Zertifizierungen finden Sie weiter unten.)
- Lehrfähigkeiten, bestätigt durch:
  - Zertifikat für Ausbildungskompetenz
    - CompTIA CTT+.
    - Essential Educate Trainer (interne Microsoft-Zertifizierung)
    - Fourth Dimension Technologies, Inc.
    - Friesen Kaye and Associates
    - Global Knowledge Instructional Skills-Zertifizierung
    - IAMCT Approved Technical Trainer (IAMCT ATT)
    - IPM (Institut für Personalberatung und Managemententwicklung)
    - Trainer Performance Monitoring und Assessment (TPMA) des Learning und Performance Institute
    - Learning Tree Train the Trainer-Programm
    - Marcom Deli
    - Microsoft Certified Trainer Instructional Skills Zertifizierung (MCT-ISC)
    - Microsoft Customer Service & Support (CSS) Trainer Certification
    - Microsoft Core Training Skills
    - New Horizons Instructional Skills-Zertifizierung
    - PrepMasters USA
    - Modern Classroom Certified Trainer (MCCT®) [nur für Hochschullehrkräfte oder -professoren]
    - Certified Virtual Educator [nur für Hochschullehrkräfte oder -professoren].
    - Sinerji Egitim
  
  oder<br/><br/>

  - Geben Sie eine Referenz an, die mindestens ein Jahr Unterrichtserfahrung nachweist. 

**Hinweis** Für eine Verlängerung Ihres Status als MCT müssen Sie zum entsprechenden Stichtag die Anforderungen an eine Verlängerung des MCT-Programms erfüllen.

## <a name="requirements-to-renew-your-mct-status"></a>Anforderungen zum Verlängern Ihres Status als MCT

- Besitz einer aktuellen, für das MCT-Programm infrage kommenden Microsoft-Zertifizierung oder -Qualifizierung (Eine Liste qualifizierender Zertifizierungen finden Sie weiter unten.)

- Erfüllung der Mindestanforderung der Unterrichtung mindestens eines Kurses, die durch [Metrics that Matter (MTM)](http://www.executiveboard.com/exbd/human-resources/metrics-that-matter/index.page) ODER ein anderes Umfragetool bestätigt wurde.

**Hinweis** Die Verlängerungsbedingungen können sich ändern.

## <a name="application-process"></a><a name="application-process"></a> Bewerbungsprozess

Wenn Sie MCT werden wollen, lesen Sie MCT Program Guide und MCT Program Agreement, um mehr über die Bedingungen der Mitgliedschaft zu erfahren.

- [MCT Program Guide](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE2XZuQ)
- [MCT Program Agreement](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE2XWWa)
- [MCT Enrollment And Renewal Guide](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE2Mf75)

Die Onlinebewerbung ist im [MCT-Registrierungstool](https://mcp.microsoft.com/Authenticate/MCT) verfügbar. Sie melden sich beim MCT-Registrierungstool mit Ihrem persönlichen Microsoft-Konto an, das mit Ihrer Microsoft-Zertifizierungs-ID (MCID) verknüpft ist.

Verwenden Sie das MCT Enrollmenttool für Folgendes:

- Sehen Sie sich die MCT-Programmanforderungen an und unterschreiben Sie die MCT-Vereinbarung (Agreement). Mit Ihrer elektronischen Signatur nehmen Sie die Bedingungen der MCT-Vereinbarung an.
- Aktualisieren Sie Ihre Profilinformationen. Ihre zur Verfügung gestellten Kontaktinformationen verwenden wir für die MCT-Kommunikation.
- Geben Sie Ihren Beschäftigungsstatus an. Wenn Sie bei einem Learning Partner oder einer Microsoft Imagine Academy beschäftigt sind, benötigen Sie die Learning Partner ID bzw. Microsoft Imagine Academy ID Ihres Arbeitgebers.
- Wählen Sie Ihre Zahlungsweise aus. Die Programmgebühren können je nach Land/Region und Beschäftigungsverhältnis unterschiedlich sein. Verwenden Sie das Nachschlage-Tool am Ende der Seite, um die Gebühren für MCT- und MCT-Alumni-Mitglieder zu finden.

Wenn Sie Microsoft Office Specialist sind und keine Microsoft-Zertifizierungs-ID (MCID) haben, können Sie eine erhalten, indem Sie sich an [customerservices@certiport.com](mailto:customerservices@certiport.com) wenden.

## <a name="mct-alumni-program"></a>MCT Alumni-Programm

Seit mehr als zwei Jahrzehnten gehören MCTs zu den Vorreitern hinsichtlich Microsoft-Technologie-Training und -Annahme. In Anerkennung ihres bedeutenden Beitrags bieten wir das MCT Alumni-Programm an. Dieses Programm richtet sich an alle MCTs, die ein gutes Ansehen bei Microsoft haben, nicht länger im Rahmen des MCT-Programms unterrichten, aber gerne ihre Beziehung zur MCT Community aufrechterhalten würden.

Anforderungen für das MCT-Alumni-Programm:

- Gegenwärtige oder vorherige Teilnahme am MCT-Programm
- Gute Beziehung zu Microsoft

Die Programmgebühren können je nach Land/Region und Beschäftigungsverhältnis unterschiedlich sein. Verwenden Sie das Nachschlage-Tool am Ende der Seite, um die Gebühren für MCT- und MCT-Alumni-Mitglieder zu finden.

## <a name="mct-alumni-benefits"></a>Leistungen für MCT-Alumnis

Als MCT Alumni-Mitglied haben Sie exklusiven Zugriff auf die MCT Alumni-Inhalte in Ihrem [Zertifizierungsdashboard](https://www.microsoft.com/learning/dashboard.aspx). Sie haben außerdem Zugang zu [MCT Central](https://www.microsoft.com/en-us/learning/mct-central.aspx), wo Sie aktuelle Informationen, Veranstaltungen und eine Vielzahl von Ressourcen für MCT Alumni finden und die Gelegenheit haben, sich mit der gesamten MCT-Community auszutauschen.

Zu den Leistungen für MCT Alumni gehören:

- Ein Rabatt auf Prüfungen, um Ihre Zertifizierung aktuell zu halten und den Wiedereinstieg in das MCT-Programm zu erleichtern.
- Rabatte auf Bücher und E-Books, die Ihnen als Referenzressourcen zur Prüfungsvorbereitung und beruflichen Weiterbildung dienen.

## <a name="moving-from-mct-alumni-to-mct"></a>Wechsel von MCT Alumni zu MCT

MCT-Alumni sind bei einer Rückkehr in eine aktive Trainertätigkeit herzlich eingeladen, sich wieder am MCT-Programm zu beteiligen.

So werden Sie wieder MCT:

- Stellen Sie sicher, dass Sie die Anforderungen des MCT-Programms erfüllen, die im Abschnitt [Anforderungen](#requirements-to-become-mct) auf dieser Seite beschrieben sind. Vergleichen Sie Ihr Transkript mit dem aktuellen Stand der für das MCT-Programm berechtigenden Microsoft Certification und -Qualifizierungen.
- Bewerben Sie sich für den Wiedereinstieg in das MCT-Programm. MCT-Alumni können entweder den Nachweis über die Zertifizierung der didaktischen Fähigkeiten erbringen, die sie zur Unterstützung ihrer früheren Bewerbung für das MCT-Programm erhalten haben, oder sie können eine Referenz vorlegen, die mindestens ein Jahr Unterrichtserfahrung bestätigt. Als Referenz können ehemalige Kunden, Vorgesetzte oder Kollegen angegeben werden.
