# Microsoft Ignite Cloud Skills Challenge 2021: Kostenlose Zertifizierungsprüfung


Vielen Dank, dass Sie die [Microsoft Ignite Cloud Skills Challenge - März 2021](https://csc.docs.microsoft.com/ignite/registration/March2021) abgeschlossen haben. Alle Teilnehmer, die mindestens eine Challenge abgeschlossen haben, haben Anspruch auf eine kostenlose Zertifizierungsprüfung gemäß [Allgemeine Geschäftsbedingungen](#terms-and-conditions), damit Sie Ihre Kenntnisse weiter verbessern und vor Arbeitgebern und Kollegen Ihr technisches Fachwissen unter Beweis stellen können.

## Einzelheiten

Die kostenlose Zertifizierungsprüfung kann ab dem 7. April 2021 eingelöst werden und ist bis zum 30. Juni 2021 gültig. Wenn Sie Ihre [berechtigte Prüfung](#eligible-exams) nicht innerhalb dieses Zeitraums planen und ablegen, können Sie Ihr Angebot nicht einlösen.

## So fordern Sie Ihre kostenlose Prüfung an

Im Folgenden finden Sie schrittweise Anleitungen, wie Sie Ihre kostenlose Prüfung beantragen können.

## FAQ

### Über das Angebot

**F: Wie viele Prüfungen kann ich mit diesem Angebot kostenlos planen?**

**A:** Sie können mit diesem Angebot eine kostenlose, berechtigte Zertifizierungsprüfung beantragen, unabhängig von der Anzahl der von Ihnen absolvierten Herausforderungen.

**F: Welche Zertifizierungsprüfungen sind für dieses Angebot berechtigt?**

**A:** Dieses Angebot beinhaltet ausgewählte Zertifizierungsprüfungen. Eine Liste der berechtigten Zertifizierungsprüfungen finden Sie im Abschnitt „[Berechtigte Prüfungen](#eligible-exams)".

**F: Kann ich dieses Angebot trotzdem beanspruchen, wenn ich mich für Microsoft Ignite registriert habe, mich aber nicht für die Cloud Skills Challenge angemeldet oder diese abgeschlossen habe?**

**A:** Nein, dieses Angebot gilt nur für diejenigen, die eine der Herausforderungen im Rahmen der Microsoft Ignite Cloud Skills Challenge (März 2021) abgeschlossen haben.

### Planen Ihrer kostenlosen Prüfung

**F: Muss ich die Prüfung am oder vor dem 30. Juni 2021 ablegen?**

**A:** Ja, Sie müssen die Prüfung am oder vor dem 30. Juni 2021 ablegen.

**F: Ich habe Probleme bei der Planung meiner kostenlosen Prüfung.Wo kann ich Unterstützung bekommen?**

**A:** Sie erhalten Hilfe, wenn Sie sich an [Microsoft Certification Support](https://trainingsupport.microsoft.com/en-us/mcp/forum) wenden. Innerhalb eines Werktages wird Ihnen ein Moderator antworten.

**F: Habe ich die Möglichkeit, die geplante Prüfung persönlich oder online abzulegen?**

**A:** Ja, Sie haben die Möglichkeit, die geplante Prüfung persönlich bei Pearson Vue in einem örtlichen Testzentrum (sofern verfügbar) oder über [online](/learn/certifications/online-exams) abzulegen. Testzentren können aufgrund von COVID-19-Einschränkungen vorübergehend geschlossen sein. Bleiben Sie auf dem Laufenden mit den aktuellen [Informationen zur Verfügbarkeit von Pearson Vue Tests](https://home.pearsonvue.com/coronavirus-update).

### Stornierungs- und Neuplanungsrichtlinie

**F: Kann ich meinen Prüfungstermin aus diesem Angebot verschieben, wenn ich muss?**

**A:** Ja, Sie können Ihren Prüfungstermin verschieben, müssen dies jedoch vor Ablauf des Angebotszeitraums tun. Weitere Informationen finden Sie unter [Stornierungs- und Neuplanungsrichtlinie](/learn/certification-exam-policies#cancellation-and-reschedule-policy).

**F: Fallen Gebühren an, wenn ich meinen Prüfungstermin verschiebe oder absage?**

**A:** Wenn Sie nicht rechtzeitig Bescheid geben, bevor Sie stornieren, könnten Sie damit Ihr kostenloses Prüfungsangebot verlieren.Wenn Sie nicht rechtzeitig Bescheid geben, bevor Sie einen neuen Termin festlegen, wird eine Gebühr erhoben. Weitere Informationen finden Sie unter [Stornierungs- und Neuplanungsrichtlinie](/learn/certifications/certification-exam-policies#cancellation-and-reschedule-policy).

**F: Was passiert, wenn ich meinen Prüfungstermin verpasst und ihn nicht verschoben oder abgesagt habe?**

**A:** Wenn Sie Ihren Prüfungstermin verpasst haben und Ihren Termin nicht gemäß der [Stornierungs- und Neuplanungsrichtlinie](/learn/certifications/certification-exam-policies#cancellation-and-reschedule-policy) verschieben oder stornieren, verfällt möglicherweise Ihr Angebot für eine kostenlose Prüfung.


### Richtlinie wiederholen

**F: Kann ich eine Prüfung kostenlos wiederholen, wenn ich das kostenlose Zertifizierungsprüfungsangebot ursprünglich nicht genutzt habe?**

**A:** Nein, das Angebot gilt nur für einen Versuch und Prüfungswiederholungen müssen separat geplant werden. Weitere Informationen zum Wiederholen von Zertifizierungsprüfungen finden Sie in unserer [Prüfungswiederholungsrichtlinie](/learn/certifications/certification-exam-policies#security-policies).
